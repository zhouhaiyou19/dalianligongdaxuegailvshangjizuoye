## 作业内容概述

### 第一题
- **生成数据**：利用`rand`函数生成1000个服从均匀分布的随机数X。

- **统计分析**：计算X和Y的方差(DX, DY)、协方差(CovXY)以及相关系数矩阵(pXY)，了解两个随机变量之间的统计关系。

### 第二题

### 第三题
